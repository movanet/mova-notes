How to Run the Rawapening Network Visualization

The visualization requires a local web server to load properly due to browser CORS restrictions with the file:// protocol.

Quick Start Options

Option 1: Python (Recommended)

If you have Python installed, navigate to the project folder and run:

Python 3:

cd "D:\Obsidian\OneNoteExport\01-Projects\12-CSIRO\Deliverables\rawapening-network-diagram"
python -m http.server 8000

Python 2:

cd "D:\Obsidian\OneNoteExport\01-Projects\12-CSIRO\Deliverables\rawapening-network-diagram"
python -m SimpleHTTPServer 8000

Then open your browser to: http://localhost:8000

Option 2: Node.js (http-server)

If you have Node.js installed:

npm install -g http-server
cd "D:\Obsidian\OneNoteExport\01-Projects\12-CSIRO\Deliverables\rawapening-network-diagram"
http-server -p 8000

Then open your browser to: http://localhost:8000

Option 3: PHP

If you have PHP installed:

cd "D:\Obsidian\OneNoteExport\01-Projects\12-CSIRO\Deliverables\rawapening-network-diagram"
php -S localhost:8000

Then open your browser to: http://localhost:8000

Option 4: VS Code Live Server Extension

  1. Open the project folder in VS Code
  2. Install the "Live Server" extension
  3. Right-click on index.html
  4. Select "Open with Live Server"

What You Should See

Once the server is running and you open the page, you should see:

  • 18 nodes (actors and policies) rendered in the center visualization area
  • Node colors indicating different tiers (red=national, blue=provincial, green=local, orange=community, gray=policies)
  • Relationship lines connecting nodes with different styles (solid, dashed, dotted)
  • Left sidebar with search and filters working
  • Right sidebar showing details when you click a node
  • Legend at the bottom explaining node and edge types

Troubleshooting

If you see a blank center area:

  • Check browser console (F12) for errors
  • Verify you're accessing via http://localhost not file://
  • Ensure data/network-data.json exists and is valid JSON

If styles look broken:

  • Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
  • Check that css/styles.css loaded properly in Network tab

If interactions don't work:

  • Check browser console for JavaScript errors
  • Verify all three JS files loaded: main.js, network-viz.js, ui-controls.js
  • Ensure D3.js CDN is accessible (check Network tab)

Browser Requirements

  • Chrome/Edge 90+
  • Firefox 88+
  • Safari 14+
  • JavaScript must be enabled

Performance

  • Initial load: 1-3 seconds
  • Network rendering: ~1 second
  • Smooth interactions with 18 nodes and 40+ edges
  • Filter updates: <100ms

Enjoy exploring the Rawapening governance network!
