🚀 Cara Menggunakan

Untuk Mahasiswa:

  1. Mulai dari Outline_Daya_Dukung_Tampung_Pencemaran.md
  2. Baca BAB_I_Pendahuluan.md untuk konteks
  3. Ikuti urutan BAB II → III → IV → V → VI
  4. Gunakan link navigasi untuk berpindah antar-bab
  5. Kerjakan studi kasus untuk latihan

Untuk Dosen:

  1. Gunakan README_Navigation.md untuk panduan mengajar
  2. Pertemuan 1-2: BAB I-III (konsep dasar)
  3. Pertemuan 3-4: BAB IV + Studi Kasus
  4. Pertemuan 5: BAB VI + Diskusi
  5. Sesuaikan alokasi waktu sesuai kebutuhan

Untuk Pembelajaran Mandiri:

  1. Ikuti urutan linear BAB I-VI
  2. Gunakan fitur search Obsidian untuk mencari topik
  3. Eksplorasi link internal untuk pemahaman lebih dalam
  4. Bookmark bagian penting

🎓 Kompetensi yang Dicapai

Setelah mempelajari materi ini, mahasiswa mampu:

✅ Memahami Konsep

  • Definisi daya dukung, daya tampung, dan pencemaran
  • Keterkaitan antara ketiga konsep
  • Dasar hukum dalam peraturan Indonesia

✅ Menganalisis Kasus

  • Mengidentifikasi pencemaran berdasarkan baku mutu
  • Menghitung daya tampung beban pencemaran
  • Menganalisis dampak terhadap daya dukung

✅ Menerapkan Hukum

  • Memahami mekanisme KLHS, baku mutu, dan perizinan
  • Mengidentifikasi konsekuensi hukum pelampauan
  • Merekomendasikan sanksi dan pemulihan yang tepat
